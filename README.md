# Art Gallery Website

Welcome to the Art Gallery website, a modern platform showcasing inspiring art collections. This project features a visually appealing design with a hero section, gallery section, and a footer. The website is designed to be responsive and engaging, providing users with a seamless experience across different devices.

## Features

- **Hero Section:** Captivating hero section with dynamic background images for desktop and mobile devices.

- **Gallery Section:** Explore distinct collections, each accompanied by a description to provide insights into the creative process.

- **Footer Section:** Informative footer section with details about the gallery's location, hours, and social media links.

## Usage

1. Clone the repository:

   ```bash
   git clone https://github.com/abhipatel9607/art_gallery
   ```

2. Open the `index.html` file in your preferred web browser.
3. Explore the hero section, gallery section, and footer for an immersive experience
4. Click on the "OUR LOCATION" button to navigate to the gallery's location page.

## Folder Structure

- `src/css/`: Contains CSS files for styling, including a reset file, utility file, and main styles.
- `src/images/`: Holds images used in the project, including hero background images, logos, and gallery images.
- `src/scripts/`: Contains the JavaScript file (script.js) for handling button interactions and navigation.
- `src/location.html`: HTML file for the gallery's location page.
